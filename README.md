# Huffman Compression Algorithm - Java

A Java implementation of Huffman Coding for text compression and decompression using Min Heap, binary tree nodes, and file handling.

## Features

- Reads text input from a file
- Counts character frequencies
- Builds a Huffman Tree using Min Heap
- Generates Huffman codes for characters
- Compresses text into encoded binary representation
- Decompresses encoded text back to the original content
- Includes sample input, compressed output, and decompressed output files

## Technologies Used

- Java
- Data Structures
- Min Heap
- Binary Trees
- File Handling
- Object-Oriented Programming

## Project Files

- `Main.java` - Main program entry point
- `HuffmanCoding.java` - Huffman coding logic
- `HuffmanNode.java` - Tree node structure
- `MinHeap.java` - Custom min heap implementation
- `FileManager.java` - File read/write handling
- `input.txt` - Sample input text
- `compressed.txt` - Compressed output
- `decompressed.txt` - Decompressed output
- `Report.docx` - Project report
